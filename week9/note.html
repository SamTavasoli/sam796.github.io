The Window Object

Every JavaScript environment has a global object. Any variables that are created in the global scope
are actually properties of this object, and any functions are methods of it. In a browser environment
the global object is the window object, which represents the browser window that contains a web page.
In this chapter we study about:
1- The Browser Object Model
2- Finding out browser information
3- Browser history
4- Controlling windows
5- Cookies
6- Timing functions

  1. The window object is the global object in a browser.
  2. Global variables are actually properties of the window object in a browser environment.
  3. alert, confirm(), and prompt() are all methods of the window object, and open dialogs that halt the execution of the program.
  4. The window.navigator object gives information about the user’s browser and operating system, although it can be unreliable
  5. The window.location object provides information about the URL of the current page.
  6. The window.history object keeps information about the pages that have been visited in the session.
  7. You can open, close, resize, and move windows (although, this doesn’t mean you should!).
  8. The window.screen object provides information about the user’s screen.
  9. document.write() is an archaic method of writing text to the document and should be avoided.
  10. Cookies can be used to store small pieces of information between requests using the document.cookie property.
  11. The window.setTimeout() method can be used to invoke a function after a set amount of time. It can be canceled using the clearTimeout() method.
  12. The window.setInterval() method can be used to repeatedly invoke a function. It can be stopped using the clearInterval() method.
  13. The window.requestAnimationFrame() method can be used to produce smooth and optimized animation by utilizing the browser's built-in graphics
    capabilities. It can be canceled using the cancelAnimationFrame() method.

HTML5 APIs

HTML5 is the latest version of the Hypertext Markup Language used to create web pages. The latest iteration is HTML 5.1,
which finally became a W3C recommendation in November 2016.

  1. HTML5.1 is the latest incarnation of the Hypertext Markup Language. It covers a variety of technologies,S
    including several APIs that are accessible using JavaScript.
  2. data- attributes help to embed custom data into a web page that can then be used to enhance the user experience with JavaScript.
  3. The Web Storage API allows key-value pairs to be stored on the user’s device in a similar way to cookies, but without the same storage restrictions.
  4. The Geolocation API allows you to access the geographic coordinates of the user’s device, as long as the user gives permission.
  5. The Web Worker API can be used to perform computationally intensive tasks in the background, which helps to avoid websites becoming unresponsive.
  6. Websockets are a new protocol for communicating over the internet, and allow real-time, two-way communication.
  7. The Notification API allows you to display notifications on the user's system.
